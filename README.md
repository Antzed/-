## 如何打卡

### 打卡前准备

[github contribute guide](https://www.dataschool.io/how-to-contribute-on-github/)

### 打卡步骤

1. 加入自己的文件夹，[]不要写进去(所有后面的步骤也都不要记进去)

   ```bash
   mkdir [你的名字/用户名等]
   ```

2. ```bash
   cd [你的文件夹名字]
   ```

3. 如果有markdown编辑器可以在编辑器里操作，如果没有就(不要键入加号)：
   ```bash
   vim [你的名字+年份].md
   ```

4. 按一下i建, 然后键入（没有晚睡理由就写null）：
   ```bash 
   [日期] [入睡时间] [晚睡理由]
   ```

5. 按下esc建，然后键入：
   ```bash
   :wq
   ```

每日一记，多记多健康

### 目标建立步骤

1. 进入你自己的文件夹

2. 遵循于于之前一样的命名逻辑
   ```bash
   vim [你的名字]+目标.md
   ```
   
3. Markdown有自己的一套task syntax，所以按下i后就可以直接键入：
   ```bash
   - [ ] [目标一] [目标截止日期/未来x时间内]
   - [ ] [目标二] [目标截止日期/未来x时间内]
   //以此类推
   ```
   

4. 重复打卡步骤第五步

鸟儿知道有虫子才会起床
